# Personal Expense Tracker API

This collection contains all API endpoints for the **Student Personal Expense Tracker**, supporting both **Web and Mobile clients.**

It includes:

- **Authentication & User** - register, login, view, and update profile.

- **Expenses** - log, update, delete, and fetch daily spending.

- **Informal Debts** - track money lent/borrowed, update status, and manage reminders.

- **Reports & Insights** - daily, weekly, and monthly summaries with category breakdowns.

- **Sync (Offline Support)** - endpoints for mobile clients to sync offline-created expenses and debts with the backend.
